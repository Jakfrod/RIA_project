Description du projet

Par David Fernandes le 08 avril 2021

Le joueur incarne un personnage qui doit passer plusieurs niveaux tout en combattant des monstres divers. Le but est très simple, ne pas mourir!. Ainsi le joueur réussissant le plus de niveau sans mourir aura le plus grand score.
